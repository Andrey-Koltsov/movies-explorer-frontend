# movies-explorer-frontend

**Frontend Сервиса, в котором можно найти фильмы по запросу и сохранить в личном кабинете.**

> Макет приложения: https://disk.yandex.ru/d/K-AfsMnaUx-WQQ

`npm run build` — Собрать проект
